# 📊 Sales Insights Data Analysis Dashboard

## 📌 Project Overview

This project analyzes sales data for a consumer electronics company to uncover valuable business insights using SQL and Tableau. The objective is to transform raw sales data into meaningful visualizations that help stakeholders monitor business performance, identify sales trends, and make data-driven decisions.

---

## 🎯 Objectives

- Analyze sales performance across different markets.
- Identify top-performing customers and products.
- Track revenue trends over time.
- Compare sales across regions.
- Build an interactive dashboard for business users.
- Support data-driven decision-making through visual analytics.

---

## 🛠️ Tech Stack

- **SQL (MySQL)** – Data extraction and analysis
- **Tableau** – Data visualization and dashboard creation

---

## 📂 Dataset

The dataset contains transactional sales data, including:

- Customers
- Products
- Markets
- Transactions
- Dates


## 📈 Dashboard Features

The Tableau dashboard includes:

- 💰 Total Revenue
- 📦 Total Sales Quantity
- 🌍 Revenue by Market
- 📅 Monthly Revenue Trend
- 🏆 Top Customers
- ⭐ Top Products
- Interactive filters for year and market selection

---

## 📊 Key Business Insights

- Identified the highest revenue-generating markets.
- Found the top customers contributing to overall sales.
- Analyzed monthly revenue fluctuations.
- Highlighted best-selling products.
- Enabled quick identification of business growth opportunities.

---

## 🚀 Business Impact

The dashboard enables decision-makers to:

- Monitor KPIs in real time.
- Identify high-performing markets and customers.
- Improve strategic planning using visual analytics.
